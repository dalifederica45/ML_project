# ML_project

The project develops a real-time kitchen safety monitoring system based on image detection using the OpenCam RT1062 platform. A lightweight classification model, trained on a custom dataset, distinguishes between dangerous and safe situations directly on-device without requiring cloud connectivity. The solution is optimized for edge execution, ensuring low latency and low power consumption for safety monitoring applications in domestic and professional kitchen environments.
